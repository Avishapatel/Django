Django E-Shopper is a full-stack **e-commerce web application** developed using the Django framework.  
This project allows users to browse products, add them to the cart, place orders, and make payments using different payment methods.

The project is built to understand real-world e-commerce workflows, backend logic, and Django MVC architecture.
